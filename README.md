 Ce Que Vous Visez

Pour cette activité, vous devez remplir tous les composants et sections fournis par LinkedIn, vérifier votre SSI et faire un message.
Instructions

    Créer ou vous connecter à votre profil LinkedIn
    Remplissez votre profil (* Image professionnelle, Titre, Résumé / À propos, En vedette, Expérience professionnelle, Compétences, Recommandations )
    Créez un post pertinent et de haute qualité.
    Suivez votre SSI avant les changements, puis après une semaine.
    Comparez les résultats, pour connaître vos prochaines étapes.

La sortie finale est un lien vers votre profil linkedin et une comparaison des résultats SSI.
